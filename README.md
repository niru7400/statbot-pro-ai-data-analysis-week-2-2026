# 📊 StatBot Pro – AI Data Analysis Dashboard

## 🚀 Overview
StatBot Pro is a lightweight AI-powered data analysis web application built using FastAPI and Python.  
It allows users to upload CSV files, ask questions about data, and generate visualizations instantly.

---

## ✨ Features

- 📂 Upload CSV datasets
- 🤖 Ask natural language questions
- 📊 Automatic data analysis (average, total, max, etc.)
- 📈 Graph generation (Bar & Pie charts)
- ⚡ Fast and lightweight (no heavy LLM dependency)
- 🛡️ Safe query handling (basic sandbox security)

---

## 🛠️ Tech Stack

- **Backend:** FastAPI
- **Frontend:** HTML, CSS (Jinja2 Templates)
- **Data Processing:** Pandas
- **Visualization:** Matplotlib
- **Server:** Uvicorn
